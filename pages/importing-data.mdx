# Importing Data into Geobase

Geobase supports importing various types of geospatial data through different tools and methods. Use the table below to find the right tool for your data format and source.

## Tool Selection Guide

| Format/Source | Recommended Tool | Alternative Tool | Guide |
|--------------|------------------|------------------|-------|
| OpenStreetMap (.osm.pbf) | osm2pgsql | - | [OSM Import Guide](./importing-data/sources/osm2pgsql) |
| Shapefile (.shp) | QGIS | ogr2ogr | [QGIS Import Guide](./guides/gis-analyst/importing-from-qgis) |
| GeoJSON (.geojson) | QGIS | ogr2ogr | [QGIS Import Guide](./guides/gis-analyst/importing-from-qgis) |
| KML (.kml) | QGIS | ogr2ogr | [Coming Soon](/importing-data/formats/gis-formats) |
| GeoPackage (.gpkg) | QGIS | ogr2ogr | [Coming Soon](/importing-data/formats/gis-formats) |
| GeoTIFF (.tiff) | QGIS | ogr2ogr | [Coming Soon](/importing-data/formats/gis-formats) |
| ESRI Geodatabase (.gdb) | QGIS | ogr2ogr | [Coming Soon](/importing-data/formats/gis-formats) |
| PostGIS | Direct Import | - | [Coming Soon](/importing-data/formats/gis-formats) |
| CSV with coordinates | QGIS | ogr2ogr | [Coming Soon](/importing-data/formats/gis-formats) |
| Overture Maps | Custom Tool | - | [Coming Soon](/importing-data/sources/sources) |

## Import Process Overview

1. Choose your tool from the table above
2. Follow the corresponding guide for detailed instructions
3. After import, verify data in the dashboard and enable in TileServer
