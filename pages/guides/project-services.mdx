# Project & organization services (API)

Placeholder roadmap for HTTP surfaces that work with Geobase: **per-project** (your provisioned instance) and **organization-wide** (shared services host).

**Project URLs** — exact bases: `geobase projects endpoints <ref>` (Geobase CLI, `cli/geobase-cli`; same merge as `geobase projects env`).

**Organization services** — same CLI table includes rows such as the routing engine; authenticate with your **organization Services API key** (Studio → organization → Services), not the project anon key.

`&#123;origin&#125;` is the project API origin (e.g. `https://&lt;ref&gt;.geobase.app`). `&#123;services_origin&#125;` is the org services host (default `https://services.geobase.app`; override in the CLI with `GEOBASE_SERVICES_API_BASE_URL`).
