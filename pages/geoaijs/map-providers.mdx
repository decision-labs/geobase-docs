## Provider Comparison

### Image Quality & Resolution
- **GeoBase**: High resolution, multispectral imagery optimized for AI analysis
- **Mapbox**: High resolution RGB satellite imagery with global coverage

### Pricing Models
- **GeoBase**: Flexible pricing options - check [geobase.app/pricing](https://geobase.app/pricing) or consult our team on [Discord](https://discord.com/invite/4susZSj4bd)
- **Mapbox**: Usage-based pricing - see [mapbox.com/pricing](https://www.mapbox.com/pricing) for current rates

### AI Model Compatibility
- **GeoBase**: Optimized for all AI tasks, especially multispectral analysis models
- **Mapbox**: Fully compatible with RGB-based AI detection and analysis models

<Callout type="info" emoji="ℹ️">
  For detailed tile server setup with GeoBase, refer to the [tile server documentation](https://docs.geobase.app/tileserver).
</Callout>

## Support & Troubleshooting

### Getting Help

For provider-specific issues and questions:

- **Documentation**: Check individual provider guides for detailed setup instructions
- **Authentication**: Verify API keys, tokens, and quota settings in your provider dashboard
- **Troubleshooting**: Consult troubleshooting sections in each provider's documentation
- **Community Support**: Join our [Discord community](https://discord.com/invite/4susZSj4bd) for real-time assistance

### Common Issues

- **Authentication Errors**: Double-check API key format and permissions
- **Rate Limiting**: Monitor usage against provider quotas and implement appropriate caching
- **Tile Loading**: Verify network connectivity and provider service status
